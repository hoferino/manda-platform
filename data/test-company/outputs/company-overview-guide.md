# Implementation Workflow

## Step 1: Review Content Document

1. Read `company-overview-content.md` thoroughly
2. Verify all facts and figures
3. Mark any corrections needed
4. Confirm narrative flow works for your buyer

## Step 2: Choose Implementation Path

**Option A: Professional Designer**
- Send `company-overview-slides.md` to designer
- Include brand guidelines
- Request first draft of 3-5 slides for approval before full build

**Option B: DIY in PowerPoint/Keynote**
- Follow slide blueprints sequentially
- Use the visual concepts as guides
- Adapt to your brand template

**Option C: AI Design Tools**
- Use blueprints as prompts for tools like Gamma, Beautiful.ai, Pitch
- Iterate on generated designs

## Step 3: Build Slides

For each slide:
1. Start with the content elements
2. Apply the visual concept layout
3. Match the color scheme to brand
4. Ensure visual hierarchy is correct
5. Add source citations as footnotes

## Step 4: Review & Iterate

- Check for consistency across slides
- Verify data matches source documents
- Test readability at presentation distance
- Get stakeholder feedback

# Design Best Practices

## Typography

| Element | Recommendation |
|---------|----------------|
| Slide Title | 28-36pt, bold, brand color |
| Action Title | 18-24pt, regular, secondary color |
| Body Text | 14-18pt, regular, dark gray |
| Table Headers | 12-14pt, bold, dark |
| Table Content | 11-13pt, regular |
| Footnotes/Sources | 8-10pt, light gray |

## Color Palette

For Vio Transgrup, consider:

| Purpose | Color | Hex | Usage |
|---------|-------|-----|-------|
| Primary | Corporate Blue | #0066CC | Headers, highlights |
| Secondary | Success Green | #00CC66 | Positive metrics, growth |
| Accent | Gold/Amber | #FFB800 | Key differentiators (TAPA) |
| Warning | Amber | #FF9900 | Risks, considerations |
| Neutral Dark | Dark Gray | #333333 | Body text |
| Neutral Light | Light Gray | #F5F5F5 | Backgrounds |
| White | White | #FFFFFF | Slide backgrounds |

## Layout Principles

1. **65/35 Rule:** Most slides use 65% content, 35% commentary
2. **White Space:** Leave minimum 10% margin on all sides
3. **Alignment:** Use consistent left/top alignment
4. **Tables:** Alternate row colors for readability
5. **Charts:** Always include axis labels and legends

## Visual Elements

| Element | When to Use |
|---------|-------------|
| Tables | Financial data, comparisons, lists |
| Bar Charts | Revenue, EBITDA, trends |
| Line Charts | Margin trends, time series |
| Pie Charts | Concentration, breakdown |
| Icons | Service categories, status indicators |
| Maps | Geographic presence |
| Timelines | Company history |
| Checklists | Certifications, requirements |

# Quality Checklist

Before finalizing, verify:

## Content Accuracy
- [ ] All numbers match knowledge.json
- [ ] Company name spelled correctly throughout
- [ ] Dates are accurate
- [ ] Percentages sum correctly
- [ ] Sources cited

## Design Consistency
- [ ] Same fonts throughout
- [ ] Colors match brand
- [ ] Headers aligned
- [ ] Tables formatted consistently
- [ ] White space adequate

## Readability
- [ ] Text readable at presentation distance
- [ ] Contrast sufficient
- [ ] Charts labeled clearly
- [ ] Legends included
- [ ] No orphaned text

## Professional Polish
- [ ] No typos
- [ ] Grammar correct
- [ ] Confidentiality notice included
- [ ] Page numbers (if printed)
- [ ] Date stamp
